# Dromdagbok

## About
A forsen fan game inspired by Yume Nikki. Explore wild worlds shaped from delirious dreams, uncover bizarre effects, and who knows - maybe *you* will find a way to bring Sebastian's mind to peace.

## Play
https://sunriser17.itch.io/dromdagbok

## Controls

- Z - confirm/interact
- X - cancel
- C - menu
- E - activate/deactivate effect
- M - mute
probably some others but i forgor
